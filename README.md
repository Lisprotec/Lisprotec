# Lisprotec — versão reconstruída para GitHub Pages

Esta versão foi refeita de raiz em HTML/CSS/JavaScript para evitar problemas de build, router e página em branco no GitHub Pages.

## Publicação

1. Criar ou usar o repositório `Lisprotec` na conta `Lisprotec`
2. Enviar todos os ficheiros desta pasta para a branch `main`
3. No GitHub ir a **Settings > Pages**
4. Em **Build and deployment**, escolher **GitHub Actions**
5. O workflow incluído publica automaticamente o site

## URL final esperada

`https://lisprotec.github.io/Lisprotec/`

## Estrutura

- `index.html` — página principal
- `privacy.html` — política de privacidade
- `terms.html` — condições de utilização
- `style.css` — estilos
- `script.js` — simulador e formulários
- `404.html` — fallback simples para Pages
